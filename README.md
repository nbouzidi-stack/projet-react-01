
# Commandes GIT

  **La toute première commande dès le lancement du projet**
    1/ git init
    2/ git add .
    3/ git commit -m "Initial commit"

  **si vous n'avez pas encore de dépôt distant**
    git remote add origin <url-de-votre-dépôt>
  **Pour envoyer vos modifications vers le dépôt distant**
    git push -u origin master
